# Nest Heroes Backend

🛡️ Backend para la gestión de héroes desarrollado con **NestJS + TypeScript**.

## 🧩 Funcionalidades
- API REST para crear, leer, actualizar y eliminar héroes.  
- Estructura modular, escalable y organizada según buenas prácticas de NestJS.

## 🛠 Tecnologías utilizadas
- **Framework**: NestJS  
- **Lenguaje**: TypeScript  
- **Testing**: (Por implementar)  

## 📦 Instalación
1. Clona el repositorio:  
   ```bash
   git clone https://github.com/Gerardoprogramer/nest-heroes-backend.git  
   cd nest-heroes-backend  

2. Instala dependencias:
   ```bash
   npm i
   
3. Ejecuta en modo desarrollo:
      ```bash
   npm run start:dev  
La API se ejecutará en http://localhost:3000.
